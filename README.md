# Gradle 构建的多模块Spring Boot 项目 Demo

- `JavaGradle`：`Java` + `Gradle` 构建的多模块`Spring Boot`项目
- `KotlinDSLGradle`：`Kotlin` + `Gradle` + `Kotlin DSL`构建的多模块`Spring Boot`项目

# 环境

- `OpenJDK 17`
- `SpringBoot 2.6.4`
- `Kotlin 1.6.10`
- `Gradle 7.4`
